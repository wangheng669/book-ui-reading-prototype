# 文本分段规则

- 输入只允许一章 HTML；先定位章节根节点，再移除 `nav/style/script/header/footer` 等非正文元素。
- 以标题、段落、列表项、图注为语义边界；不得摘要、改写、合并句子或补全残句。
- `text` 必须是 `sourceHtml` 去标签后的等义文本；空白规范化不视为改写。
- block id 按章节内稳定顺序生成：`chapter-01-block-001`。
- 标题可用于 chapter 元数据，不与正文段落重复生成 block。
- 如果一个 DOM 段落包含明显的小标题和正文，第一阶段仍保持一个 block，留给人工审核决定是否拆分。
