# Reading Prototype

React + Vite 单页阅读原型。正文和互动均来自 `content-pipeline/output/`，没有把章节内容写入组件。

## 已实现

- 第1/2章切换与连续滚动阅读。
- 段落级焦点判断：第2章每个block驱动右侧companion。
- 对比、推论、时间线、指标、因果、证据与结论视图。
- 推论判断、时间尺度、购买力变量和逐步因果链深度互动。
- 原文数字锚点、知识点主动回忆和两阶段章节复盘。
- 响应式布局；窄屏右侧画布降级为按需浮层。
- 调试面板默认隐藏，仅开发环境使用 `?debug=1` 显示。

## 运行

```bash
pnpm install
pnpm dev
```

生产构建：

```bash
pnpm build
pnpm preview
```
